# Fitzone Fitness Studio

A responsive fitness studio website built using HTML, CSS, and JavaScript.

## Features
- Fully responsive (mobile-friendly)
- Modern UI design
- BMI calculator
- Multiple pages (Home, About, Services, Trainers, Gallery, Contact)
- Deployed on Netlify

## Tech Stack
- HTML5
- CSS3
- JavaScript
